# expectJsonLength

Asserts on the length of JSON array objects.

# Syntax

expectJsonLength(length)
expectJsonLength(path, length)
1
2

# Usage

# âœ… Correct Usage

await spec()
  .get('/api/users')
  .expectJsonLength(1);

# Arguments

# > length (number)

Expected json object length.

# > path (string)

Json path. See json-query (opens new window) for more usage details.
